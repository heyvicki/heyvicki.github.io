Get Started

To get started, enable Vicki on both Alexa and on your web browser with your Amazon account.

Enable Alexa skill & Install browser extension


After your Amazon account is linked on both places, just say "Alexa, ask web browser to browse amazon".

To see all the things you can ask Vicki, click "manage scripts" on the browser extension.

Scripts for Vicki

Search the web for user created VickiScripts. Here are some good places to start.

Write your own script

Write javascript to answer requests from Alexa on your browser. Here is an example. 

        /// ==VickiScript==
/// @name Hello World
/// @utterance Tell $invocation hello
/// ==/VickiScript==

$vs.listen(async (request, response) => {
  // A alexa request came in!

  if (request.utterance !== "hello") {
    // We can't answer this request.
    return;
  }

  // Open hello world wiki page on a new tab.
  const url = "https://en.wikipedia.org/wiki/%22Hello,_World!%22_program";
  await browser.tabs.create({ url });
  
  // Make Alexa say "Hello World"
  response.say("Hello World").send();
});
